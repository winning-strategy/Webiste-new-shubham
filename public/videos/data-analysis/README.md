# Data Analysis Agent Videos

This directory contains video demonstrations for the Data Analysis Agent product page.

## Required Video Files

1. `data-integration.mp4` - Data integration from multiple sources with cleaning demo
2. `exploratory-analysis.mp4` - Exploratory data analysis with EDA and visualizations demo
3. `statistical-modeling.mp4` - Statistical modeling and regression analysis demo
4. `pattern-recognition.mp4` - Pattern recognition and anomaly detection demo
5. `data-visualization.mp4` - Data visualization and business dashboard creation demo
6. `pipeline-engineering.mp4` - Pipeline engineering and automation demo

## Video Specifications

- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Aspect Ratio**: 16:9
- **Duration**: 30-90 seconds per demo
- **File Size**: 5-15MB per video
- **Codec**: H.264
- **Audio**: AAC (if included)

## Recommended Tools

- **Screen Recording**: Focusee, Screen Studio, OBS Studio
- **Compression**: FFmpeg, HandBrake
- **Editing**: DaVinci Resolve (free), Adobe Premiere Pro

## Content Guidelines

Each video should demonstrate:
- Clear prompt input for data analysis tasks
- Jupyter notebook-like interface with code cells
- Automatic code execution and visualization
- Business insights generation
- API integration capabilities
- Statistical analysis and modeling

## File Naming Convention

Use descriptive filenames that match the video content:
- `data-integration.mp4` - Data integration and cleaning
- `exploratory-analysis.mp4` - Exploratory data analysis
- `statistical-modeling.mp4` - Statistical modeling and regression
- `pattern-recognition.mp4` - Pattern recognition and anomaly detection
- `data-visualization.mp4` - Data visualization and dashboards
- `pipeline-engineering.mp4` - Pipeline engineering and automation
