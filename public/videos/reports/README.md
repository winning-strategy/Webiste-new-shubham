# Report Generation Agent Videos

This directory contains video demonstrations for the Report Generation Agent product page.

## Required Video Files

1. `exec-summary.mp4` - Executive summary generation from lengthy reports demo
2. `market-research.mp4` - Market research report creation with data synthesis demo
3. `performance-analysis.mp4` - Performance analysis report with KPIs and benchmarks demo
4. `compliance-reports.mp4` - Compliance and regulatory report generation demo
5. `project-status.mp4` - Project status report consolidation demo
6. `client-proposals.mp4` - Client proposal generation with templates demo

## Video Specifications

- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Aspect Ratio**: 16:9
- **Duration**: 30-90 seconds per demo
- **File Size**: 5-15MB per video
- **Codec**: H.264
- **Audio**: AAC (if included)

## Recommended Tools

- **Screen Recording**: Focusee, Screen Studio, OBS Studio
- **Compression**: FFmpeg, HandBrake
- **Editing**: DaVinci Resolve (free), Adobe Premiere Pro

## Content Guidelines

Each video should demonstrate:
- Clear prompt input
- AI processing and analysis
- Report generation with proper formatting
- Professional output with charts/tables
- Export capabilities

## File Naming Convention

Use descriptive filenames that match the video content:
- `exec-summary.mp4` - Executive summary generation
- `market-research.mp4` - Market research reports
- `performance-analysis.mp4` - Performance analysis
- `compliance-reports.mp4` - Compliance reports
- `project-status.mp4` - Project status updates
- `client-proposals.mp4` - Client proposal generation
